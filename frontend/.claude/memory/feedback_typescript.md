Always use TypeScript (.tsx/.ts) for frontend code in this project — never plain JavaScript (.jsx/.js).

**Why:** User explicitly requested TypeScript when building the search engine frontend.

**How to apply:** Use .tsx for React components, .ts for utility/API files. Include proper type definitions, interfaces for API responses, and tsconfig.json. Add @types/* packages as needed.
